# Predictive Forecasting of Care Load & Placement Demand

## Project Overview

This project focuses on forecasting **care load and placement demand** for the **HHS Unaccompanied Alien Children (UAC) Program** using time-series forecasting and machine learning techniques.

The system analyzes historical care data to predict future demand and help organizations plan resources more efficiently.

The project combines statistical forecasting models and machine learning to improve prediction accuracy and provide insights through an interactive dashboard built using **Streamlit**.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Statsmodels
- Streamlit

---

## Forecasting Models Used

This project implements three different forecasting models:

### 1. ARIMA (AutoRegressive Integrated Moving Average)

A statistical time-series model used to understand and predict future values based on historical trends and patterns.

### 2. ETS (Error Trend Seasonality)

An exponential smoothing method designed for forecasting time-series data with trend and seasonal components.

### 3. Random Forest Regressor

A machine learning model that uses multiple decision trees to capture complex relationships in the dataset and improve prediction accuracy.

These models allow comparison of forecasting performance and help generate reliable demand predictions.

---

## Project Structure

```
hhs-care-demand-forecast

│
├── Care_Load_Forecasting.ipynb
│   Data preprocessing, exploratory analysis and model training
│
├── main.py
│   Streamlit dashboard for visualization and forecasting results
│
├── HHS_Unaccompanied_Alien_Children_Program.csv
│   Main dataset used for training the models
│
├── forecast_predictions.csv
│   Forecasted demand results generated by the models
│
├── kpi_table.csv
│   Key performance metrics used for analysis
│
├── requirements.txt
│   List of required Python libraries
│
└── README.md
    Project documentation
```

---

## Features

- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Time-series forecasting using ARIMA and ETS  
- Machine learning prediction using Random Forest  
- Model evaluation and KPI analysis  
- Interactive Streamlit dashboard  
- Visualization of care demand trends and forecasts  

---

## How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/amatussabur21-dotcom/hhs-care-demand-forecast.git
```

### 2. Navigate to the project folder

```
cd hhs-care-demand-forecast
```

### 3. Install required libraries

```
pip install -r requirements.txt
```

### 4. Run the Streamlit application

```
python -m streamlit run main.py
```

The Streamlit dashboard will automatically open in your web browser.

The system generates:

- Forecasted care demand values  
- KPI performance metrics  
- Visual trend analysis  
- Interactive forecasting dashboard
